Git Commands Reference

Copy and run these commands in your terminal to trigger the CI/CD pipeline

1. Make Changes & Commit

git add .
git commit -m "Update: your changes"

Stage and commit your changes locally

2. Push to GitHub (Triggers Pipeline)

git push origin main

Push to main branch - this automatically triggers the CI/CD pipeline

3. Check Pipeline Status

gh run list

View recent workflow runs (requires GitHub CLI)

Create Feature Branch

git checkout -b feature/new-feature

Push Feature Branch

git push origin feature/new-feature

View Workflow Logs

gh run view --log

Trigger Manual Workflow

gh workflow run deploy.yml

Check Git Status

git status

View Recent Commits

git log --oneline -5

Force Push (Use Carefully!)

git push origin main --force

Warning: This overwrites remote history

Reset to Last Commit

git reset --hard HEAD~1

Warning: This discards uncommitted changes
